API Reference

Core

cf.create_group Creates a TileDB group with arrays at relative locations inside the group.
cf.open_group_array Opens an array in a group either by specifying the name of the array or the name
cf.ArrayMetadata Class for accessing array-related metadata from a TileDB metadata object.
cf.AttrMetadata Metadata wrapper for accessing attribute metadata.
cf.DimMetadata Metadata wrapper for accessing dimension metadata.
cf.DataspaceCreator Creator for a group of arrays that satify the CF Dataspace Convention.

NetCDF Support

cf.from_netcdf Converts a NetCDF input file to nested TileDB CF dataspaces.
cf.NetCDF4ConverterEngine Converter for NetCDF to TileDB using netCDF4.
cf.netcdf_engine.NetCDF4CoordToDimConverter Converter for a NetCDF variable/dimension pair to a TileDB dimension.
cf.netcdf_engine.NetCDF4DimToDimConverter Converter for a NetCDF dimension to a TileDB dimension.
cf.netcdf_engine.NetCDF4ScalarToDimConverter Converter for NetCDF scalar (empty) dimensions to a TileDB Dimension.
cf.netcdf_engine.NetCDF4ArrayConverter Converter for a TileDB array from a collection of NetCDF variables.
cf.netcdf_engine.NetCDF4DomainConverter Converter for NetCDF dimensions to a TileDB domain.
cf.netcdf_engine.NetCDF4ToDimConverter Converter from NetCDF to a TileDB dimension in a NetCDF4ArrayConverter
cf.netcdf_engine.NetCDF4VarToAttrConverter Converter for a NetCDF variable to a TileDB attribute.

Xarray Support

cf.from_xarray Creates a TileDB group and copies all data from an xarray dataset.
cf.create_group_from_xarray Creates a TileDB group and arrays from an xarray dataset.
cf.copy_data_from_xarray Copies data from an xarray dataset to a TileDB group.
cf.copy_metadata_from_xarray Copies metadata from an xarray dataset to a TileDB group.
